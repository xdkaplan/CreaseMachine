# Definition of Done

Terse. A feature is **done** when:

- **Replayable** — every command is replayable from the journal.
- **Readable & concise** — the journal is human-readable and concise; use a `#` comment only when the user's action can't be inferred from the deltas alone.
- **Regen / dependency tree functional** — the Real / Transient / Ephemeral relationships are functional and test-protected.
